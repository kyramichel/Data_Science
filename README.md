
## Data_Science (Python)

This repository contains foundational data science projects and experiments developed in Python, covering data analysis, statistical methods, and introductory machine learning techniques.

This was one of my early public repositories and contributed to recognition within the GitHub community.

---

* **Exploratory Data Analysis (EDA):**
  Data cleaning, visualization, and pattern discovery

* **Statistical Methods:**
  Descriptive statistics and basic inferential techniques

* **Introductory Machine Learning:**
  Baseline predictive models and standard ML workflows

* **Python for Data Workflows:**
  End-to-end data processing and analysis pipelines
